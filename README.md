# lexicase-6870

LexiCase is a modern legal tech platform designed to streamline case management and document automation for solo practitioners and small law firms. It provides an intuitive dashboard to organize cases, track deadlines, generate standard legal documents, and securely store client information, all in one centralized, cloud-based system. The MVP focuses on core workflows to reduce administrative overhead and improve efficiency for legal professionals.

## Tech Stack
Next.js, Node.js, PostgreSQL, Tailwind CSS

## Setup
Run `docker-compose up` to start the infrastructure.
